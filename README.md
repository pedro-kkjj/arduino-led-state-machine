# arduino-led-state-machine
Projeto desenvolvido no Tinkercad para praticar programação embarcada utilizando Arduino.

## Funcionalidades

- Controle de dois LEDs por botão.
- Clique curto altera a fase.
- Clique longo altera o modo.
- Temporização utilizando `millis()`.
- Máquina de estados para controle das fases e modos.
- Código organizado utilizando funções.

## Tecnologias

- Arduino
- C++
- Tinkercad

## Conceitos praticados

- Máquina de estados
- Programação sem `delay()`
- Modularização com funções
- Controle de tempo utilizando `millis()`
- Leitura de botão
- Organização de código

## Estrutura do código

- `detectarBotao()`
- `definirLedsAtivos()`
- `executarModo()`
- `piscarSeAtivo()`
- `ligarSeAtivo()`
